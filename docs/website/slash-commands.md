Use slash commands in the chat input for quick actions. Type `/` to see autocomplete suggestions.

| Command | Description |
|---|---|
| `/help` | List all available commands |
| `/memory` | View stored preferences and memory |
| `/connector add\|list\|remove\|search` | Manage MCP connectors |
| `/workflow list` | List available workflows |
| `/config list` | Show config generations |
| `/cost` | Show token usage and cost breakdown |
| `/model list` | List available LLM models |
