# Product Launch Checklist

## Phase 1: Pre-Launch
- [ ] Finalize landing page copy
- [ ] Set up analytics tracking (Plausible)
- [ ] Create demo video (2 min walkthrough)
- [ ] Write announcement blog post
- [ ] Prepare social media posts (Twitter, LinkedIn, HN)

## Phase 2: Launch Day
- [ ] Publish blog post at 9:00 AM CET
- [ ] Post on Hacker News
- [ ] Share on Twitter/X
- [ ] Send newsletter to early adopters
- [ ] Monitor error logs and metrics

## Phase 3: Post-Launch
- [ ] Respond to HN comments
- [ ] Collect user feedback
- [ ] Fix critical bugs within 24h
- [ ] Plan v0.4 based on feedback

## Key Messages
- "Your AI, your machine, your data"
- "No cloud dependency — runs locally"
- "Starts simple, grows with you"
