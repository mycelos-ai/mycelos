# Product Review — April 11

## Agenda
1. Q1 metrics review
2. Feature prioritization for Q2
3. Customer feedback themes

## Q1 Results
- **Active users:** 2,400 (+34% QoQ)
- **Retention (30-day):** 68%
- **NPS:** 42 (up from 35)
- **Support tickets:** down 22% after knowledge base launch

## Q2 Feature Priorities
1. **Workflow templates** — most requested feature (47 votes)
2. **Team sharing** — enterprise customers asking
3. **Mobile app** — v1 with chat + knowledge read access
4. **Integrations** — Notion, Linear, GitHub

## Customer Feedback Themes
- "Love the local-first approach, but need team sync"
- "Workflows are powerful but hard to discover"
- "Knowledge base search could be smarter"
- "Want to use it on my phone"

## Decisions
- Workflow templates → ship in v0.4
- Team sharing → design phase, ship in v0.5
- Mobile → start prototype in May
